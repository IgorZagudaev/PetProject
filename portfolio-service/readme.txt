Описание:
    Сервис по операциям со аккаунтом и его счётом (Доступны при валидном токене).
    Операции:
        создание аккаунта
        изменение баланса - пессмистичная блокировка, ретрай, рекавер
        получение списка аккаунтов

Эндпоинты:
    POST  /api/v1/accounts/create,
    PATCH /api/v1/accounts/{accountId}/balance,
    GET   /api/v1/accounts

Хотелки:

    добавить API-key для внутренних сервисов
